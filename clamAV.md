ClamAV (антивирус)
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование ПО: ClamAV
Краткое описание
ClamAV — это антивирусное программное обеспечение с открытым исходным кодом, предназначенное для обнаружения различных видов вредоносного ПО, включая вирусы, трояны, черви и эксплойты. Он широко используется на почтовых серверах, файловых серверах и рабочих станциях под управлением Linux. ClamAV обновляется несколько раз в день, обеспечивая актуальную защиту от новых угроз. Включает как консольную утилиту для сканирования, так и графический интерфейс ClamTk для удобного использования.
Запуск программы
•	Проверка версии: clamscan --version
•	Обновление вирусных баз: sudo freshclam
•	Сканирование файла: clamscan /путь/к/файлу
•	Сканирование папки: clamscan -r /путь/к/папке
•	Графический интерфейс: clamtk
Запланированный выход
•	Для clamscan: программа завершается автоматически после сканирования
•	Для clamtk: «Файл» → «Выход» или закрыть окно
•	Для freshclam: завершается автоматически после обновления
Основные функции
1.	Сканирование отдельных файлов и папок на наличие вирусов
2.	Рекурсивное сканирование целых каталогов
3.	Автоматическое обновление вирусных баз через freshclam
4.	Карантин для подозрительных файлов
5.	Сканирование архивов (ZIP, RAR, GZIP и др.)
6.	Интеграция с почтовыми серверами для проверки вложений
7.	Графический интерфейс для упрощения работы
Основные приемы работы
1. Обновление вирусных баз
Перед сканированием обязательно обновите базы:
sudo freshclam
Это загрузит свежие сигнатуры вирусов. Если базы не обновлять, многие новые угрозы будут пропущены.
2. Сканирование файла
clamscan /home/user/downloads/file.zip
Программа покажет результат: будет найден вирус или нет.
3. Сканирование всей папки (рекурсивно)
clamscan -r /home/user/downloads/
Ключ -r означает рекурсивный обход всех вложенных папок.
4. Сканирование с выводом зараженных файлов только на экран
clamscan -r --infected /home/user/
Будут показаны только файлы, в которых обнаружены вирусы.
5. Сканирование с сохранением лога в файл
clamscan -r /home/user/ -l scan_result.log
Результат запишется в файл scan_result.log, который можно потом просмотреть.
6. Помещение зараженных файлов в карантин
clamscan -r --move=/home/user/quarantine /home/user/downloads/
Все зараженные файлы будут перемещены в папку карантина.
7. Удаление зараженных файлов
clamscan -r --remove /home/user/downloads/
ВНИМАНИЕ: зараженные файлы будут удалены безвозвратно. Используйте с осторожностью.
8. Запуск графического интерфейса ClamTk
clamtk
Откроется окно с кнопками для выбора папки сканирования, обновления баз и настройки параметров. Интерфейс интуитивно понятен: выберите папку и нажмите «Scan».
9. Автоматическое сканирование по расписанию
Можно настроить задание в cron для автоматического сканирования, например, каждый день в 2 часа ночи:
sudo crontab -e
Добавить строку:
0 2 * * * clamscan -r --move=/var/quarantine /home/ 2>&1 >> /var/log/clamav/scan.log
10. Проверка статистики
После сканирования ClamAV выводит статистику:
•	Количество просканированных файлов
•	Количество обнаруженных угроз
•	Время выполнения
Это полезно для отчетности.
