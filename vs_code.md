VS Code (универсальная среда разработки)
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование ПО: Visual Studio Code
Краткое описание
Visual Studio Code (VS Code) — это легкий, но мощный редактор кода, который стал стандартом для веб-разработчиков и программистов на многих языках. Он бесплатен, работает на всех платформах и обладает огромным количеством расширений, которые добавляют поддержку языков, отладчиков, линтеров и инструментов. VS Code сочетает скорость простого редактора с возможностями полноценной IDE: автодополнение, подсветка синтаксиса, рефакторинг, встроенный терминал, отладка и интеграция с Git.
Запуск программы
•	Из терминала: code
•	Из меню: «Приложения» → «Разработка» → «Visual Studio Code»
•	Из командной строки с указанием папки: code . (открыть текущую папку) или code /путь/к/папке
Запланированный выход
•	Горячие клавиши: Ctrl+Q (закрыть) или Ctrl+W (закрыть текущий файл)
•	Через меню: «File» → «Exit»
•	Если есть несохраненные файлы, программа запросит подтверждение
Основные функции
1.	Подсветка синтаксиса для сотен языков программирования
2.	Интеллектуальное автодополнение кода (IntelliSense)
3.	Интеграция с Git для управления версиями
4.	Встроенный терминал для выполнения команд без переключения окон
5.	Расширяемость — тысячи бесплатных плагинов
6.	Встроенный отладчик с точками останова
7.	Рефакторинг кода (переименование, перемещение функций)
8.	Работа с несколькими файлами и проектами в одном окне
9.	Поддержка линтеров и форматтеров для поддержания чистоты кода
Основные приемы работы
1. Открытие проекта
Запустите VS Code. Нажмите «Open Folder» и выберите папку с вашим проектом. После открытия все файлы проекта появятся в левой панели (Explorer). Можно также открыть терминал и ввести code . в папке проекта — VS Code откроется автоматически с этой папкой.
2. Создание и сохранение файла
Нажмите Ctrl+N для создания нового файла. Нажмите Ctrl+S для сохранения — выберите имя и расширение (например, index.html, app.js, style.css). После сохранения редактор автоматически включит подсветку синтаксиса.
3. Поиск и замена
Ctrl+F — найти в текущем файле. Ctrl+Shift+F — найти по всему проекту. Ctrl+H — замена в текущем файле.
4. Команды в терминале
Откройте встроенный терминал: Ctrl+``` (клавиша с тильдой). В терминале можно выполнять команды без переключения окон:npm install,git status,python script.py,node server.js` и т.д.
5. Установка расширений
Нажмите на иконку «Extensions» (квадратики) на левой панели или Ctrl+Shift+X. В строке поиска введите название расширения, например:
•	«Python» — поддержка Python
•	«JavaScript (ES6)» — поддержка JavaScript
•	«Prettier» — автоматическое форматирование
•	«ESLint» — проверка кода на ошибки
•	«GitLens» — удобная работа с Git
Нажмите «Install» на нужном расширении.
6. Использование IntelliSense (автодополнение)
Начните печатать код — редактор подскажет возможные варианты. Например, в Python после ввода import появится список доступных модулей. В JavaScript после ввода точки у объекта появятся все его свойства и методы. Для вызова подсказки вручную нажмите Ctrl+Space.
7. Отладка кода
Слева на панели нажмите иконку «Run and Debug» (жучок) или Ctrl+Shift+D. Нажмите «Create a launch.json file» и выберите среду (Node.js, Python, Chrome). Установите точку останова, кликнув слева от номера строки. Запустите отладку — выполнение приостановится на точке останова, и вы сможете просматривать значения переменных.
8. Использование Git
VS Code интегрирован с Git. Слева на панели есть иконка «Source Control» (ветвь). Там отображаются все изменения в файлах. Вы можете добавлять файлы в коммит, писать сообщение и нажимать галочку для коммита. В меню «...» можно выбрать «Push» или «Pull».
9. Форматирование кода
Установите расширение Prettier. Для форматирования всего файла нажмите Shift+Alt+F. Или настройте автоматическое форматирование при сохранении: в настройках Ctrl+, включите «Editor: Format On Save».
10. Навигация по коду
•	Ctrl+P — быстрый переход к файлу (введите имя файла)
•	Ctrl+Shift+O — переход к символу в текущем файле (функции, классы, переменные)
•	F12 — переход к определению функции/класса
•	Alt+Left / Alt+Right — навигация по открытым файлам
11. Мультикурсор
Зажмите Alt и кликайте по нужным местам — появится несколько курсоров. Это удобно для одновременного редактирования одинаковых строк. Ctrl+D — выделить следующее совпадение текущего слова.
12. Настройка пользовательских сниппетов
В настройках (Ctrl+,) выберите «User Snippets». Создайте сниппет для языка, например, для HTML:
"HTML template": {
    "prefix": "html5",
    "body": [
        "<!DOCTYPE html>",
        "<html lang=\"ru\">",
        "<head>",
        "  <meta charset=\"UTF-8\">",
        "  <title>$1</title>",
        "</head>",
        "<body>",
        "  $2",
        "</body>",
        "</html>"
    ],
    "description": "HTML5 шаблон"
}
Теперь при вводе html5 и нажатии Tab вставится шаблон.
