1. GIMP (графический редактор)
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование ПО: GIMP (GNU Image Manipulation Program) версия 2.10
Краткое описание
GIMP — это бесплатный растровый графический редактор, который часто используют как альтернативу Adobe Photoshop. Программа подходит для обработки фотографий, создания коллажей, рисования, ретуши и подготовки изображений для веб-сайтов. Поддерживает большинство популярных форматов (JPEG, PNG, GIF, TIFF, PSD) и позволяет работать со слоями, масками, фильтрами и эффектами. Благодаря встроенной поддержке скриптов и плагинов, функциональность можно расширять под любые задачи.
Запуск программы
•	Из терминала: gimp
•	Из меню: «Приложения» → «Графика» → «GIMP»
•	Двойным кликом по ярлыку на рабочем столе (если создан)
Запланированный выход
•	Горячие клавиши: Ctrl+Q
•	Через меню: «Файл» → «Выход»
•	При наличии несохраненных изменений программа предложит сохранить их перед закрытием
Основные функции
1.	Создание и редактирование изображений с настройкой размера, разрешения и цветового режима
2.	Работа со слоями — добавление, удаление, изменение прозрачности, режимов наложения
3.	Применение фильтров и эффектов (размытие, резкость, шум, искажение, художественные фильтры)
4.	Коррекция цвета и тона с помощью уровней, кривых, баланса белого, насыщенности
5.	Инструменты выделения любой формы — прямоугольное, эллиптическое, лассо, по цвету
6.	Работа с текстом — добавление надписей с настройкой шрифта, размера, цвета
7.	Экспорт в различные форматы с настройкой качества и сжатия
Основные приемы работы
1. Создание нового изображения
Нажмите Ctrl+N или выберите «Файл» → «Создать». Укажите ширину и высоту в пикселях (например, 1920×1080), разрешение (72 DPI для экрана, 300 DPI для печати), цветовой режим (RGB — стандарт). Нажмите «ОК».
2. Открытие существующего файла
Используйте Ctrl+O или «Файл» → «Открыть». Найдите файл на диске и нажмите «Открыть». Изображение загрузится в рабочую область.
3. Работа со слоями
Откройте панель слоев: Ctrl+L. Для создания нового слоя нажмите значок «+» внизу панели или Shift+Ctrl+N. Перетаскивайте слои для изменения порядка — верхние слои перекрывают нижние. Меняйте прозрачность ползунком «Непрозрачность».
4. Выделение области
Выберите инструмент выделения (прямоугольное — R, эллиптическое — E, свободное — F). Обведите нужную область. Для добавления к выделению зажмите Shift, для вычитания — Ctrl. Инвертировать выделение можно комбинацией Ctrl+I.
5. Сохранение и экспорт
Для сохранения проекта со всеми слоями используйте Ctrl+S — файл сохранится в формате .xcf. Для экспорта в готовый формат (JPEG, PNG) используйте Shift+Ctrl+E или «Файл» → «Экспортировать как». В открывшемся окне выберите формат, настройте качество и нажмите «Экспорт».
6. Отмена действий
Ctrl+Z — отменить последнее действие. Shift+Ctrl+Z — вернуть отмененное действие. История доступна в меню «Окна» → «История».
