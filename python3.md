5. Python 3 (разработка)
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование ПО: Python 3.10+
Краткое описание
Python — это высокоуровневый интерпретируемый язык программирования, который ценится за простой и понятный синтаксис. Он широко применяется в веб-разработке (Django, Flask), анализе данных (pandas, NumPy), машинном обучении (scikit-learn, TensorFlow), автоматизации, научных вычислениях и создании скриптов. Огромная экосистема библиотек и активное сообщество делают Python универсальным инструментом для задач любого масштаба.
Запуск программы
•	Интерактивный режим: python3 (после запуска можно вводить команды построчно)
•	Запуск скрипта: python3 script.py
•	Запуск с модулем: python3 -m имя_модуля
•	Выход из интерактивного режима: exit() или Ctrl+D
Запланированный выход
•	В интерактивном режиме: exit() или Ctrl+D
•	При выполнении скрипта: программа завершается автоматически после выполнения последней инструкции или при вызове sys.exit()
•	Принудительная остановка выполняющегося скрипта: Ctrl+C
Основные функции
1.	Обработка и анализ данных с использованием библиотек pandas, NumPy
2.	Визуализация данных с помощью matplotlib, seaborn, plotly
3.	Создание веб-приложений (Flask, Django)
4.	Написание скриптов для автоматизации задач (обработка файлов, парсинг, работа с API)
5.	Математические вычисления и научные расчеты
6.	Машинное обучение (scikit-learn) и глубокое обучение (PyTorch, TensorFlow)
7.	Работа с базами данных через ORM (SQLAlchemy) или драйверы
Основные приемы работы
1. Запуск интерактивного режима
В терминале введите python3. Появится приглашение >>>. Можно вводить команды и видеть результат сразу:
>>> 2 + 2
4
>>> print("Hello")
Hello
Выйти можно с помощью exit() или Ctrl+D.
2. Создание и запуск скрипта
Создайте файл hello.py:
name = input("Введите ваше имя: ")
print(f"Привет, {name}!")
Запустите: python3 hello.py. Программа запросит имя и выведет приветствие.
3. Установка библиотек через pip
Установка библиотеки для работы с таблицами:
pip3 install pandas
Установка библиотеки для визуализации:
pip3 install matplotlib
Установка нескольких библиотек сразу:
pip3 install numpy scipy scikit-learn
4. Импорт библиотек и использование
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# Создание DataFrame
data = {'Имя': ['Анна', 'Петр', 'Мария'], 'Возраст': [25, 30, 28]}
df = pd.DataFrame(data)
print(df)

# Построение графика
x = np.linspace(0, 10, 100)
y = np.sin(x)
plt.plot(x, y)
plt.title('График синуса')
plt.show()
5. Работа со списками и циклами
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num * 2)

# Генератор списка
squares = [x**2 for x in range(10)]
print(squares)
6. Работа с файлами
Чтение из файла:
with open('file.txt', 'r') as f:
    content = f.read()
    print(content)
Запись в файл:
with open('output.txt', 'w') as f:
    f.write('Hello, world!')
7. Создание функции
def greet(name, greeting="Привет"):
    return f"{greeting}, {name}!"

print(greet("Алексей"))
print(greet("Мария", "Здравствуйте"))
8. Использование виртуального окружения (рекомендуется для проектов)
Создание виртуального окружения:
python3 -m venv myenv
Активация:
•	Linux/Mac: source myenv/bin/activate
•	Windows: myenv\Scripts\activate
Установка пакетов внутри окружения (без влияния на систему):
pip install flask
Деактивация: deactivate
9. Использование аргументов командной строки
Создайте файл args.py:
import sys
print("Передано аргументов:", len(sys.argv))
print("Аргументы:", sys.argv)
Запуск: python3 args.py один два три
Вывод: Передано аргументов: 4, Аргументы: ['args.py', 'один', 'два', 'три']
