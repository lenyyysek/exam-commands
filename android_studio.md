6. Android Studio (мобильная разработка)
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование ПО: Android Studio (актуальная стабильная версия)
Краткое описание
Android Studio — это официальная интегрированная среда разработки (IDE) для создания приложений под операционную систему Android. Она основана на IntelliJ IDEA и включает в себя всё необходимое для разработки: редактор кода с подсветкой синтаксиса и автодополнением, визуальный редактор интерфейсов, встроенный эмулятор устройств, инструменты отладки и профилирования. Android Studio поддерживает языки Java, Kotlin и C++, а также Gradle для автоматизации сборки. Это основной инструмент, который используют профессиональные Android-разработчики.
Запуск программы
•	Из терминала (если установлена через snap): android-studio
•	Из меню приложений: «Приложения» → «Разработка» → «Android Studio»
•	Из папки установки: ./android-studio/bin/studio.sh
Запланированный выход
•	Через меню: «File» → «Exit»
•	Горячие клавиши: Ctrl+Q
•	При наличии несохраненных изменений программа спросит, нужно ли сохранить их перед закрытием
Основные функции
1.	Создание проектов с шаблонами для различных типов приложений (Empty Activity, Bottom Navigation, Tabbed Activity и др.)
2.	Визуальный редактор макетов (Layout Editor) — создание интерфейсов методом drag-and-drop
3.	Встроенный эмулятор Android для тестирования приложений на разных версиях ОС и устройствах
4.	Отладка кода с точками останова, пошаговым выполнением и просмотром значений переменных
5.	Система сборки Gradle — управление зависимостями, версиями библиотек, настройками сборки
6.	Инструменты профилирования производительности (CPU, Memory, Network)
7.	Интеграция с системами контроля версий (Git, GitHub)
8.	Поддержка Kotlin, Java и C++ в одном проекте
Основные приемы работы
1. Создание нового проекта
Откройте Android Studio. На стартовом экране выберите «New Project». Выберите шаблон активности (например, «Empty Views Activity»). Укажите имя проекта, пакет, язык (Java или Kotlin) и минимальную версию Android SDK. Нажмите «Finish». Проект будет создан и загружен.
2. Структура проекта
Основные папки в проекте:
•	app/src/main/java — исходный код на Java/Kotlin
•	app/src/main/res — ресурсы (макеты, картинки, строки, стили)
o	layout — XML-файлы интерфейса
o	drawable — изображения
o	values — строки, цвета, темы
•	app/src/main/AndroidManifest.xml — описание приложения (разрешения, активности, сервисы)
3. Запуск эмулятора
Нажмите на иконку «Device Manager» (иконка смартфона) на панели инструментов. Выберите «Create Virtual Device» и выберите устройство (например, Pixel 6). Скачайте системный образ (желательно последнюю стабильную версию). Нажмите «Next» и «Finish». Для запуска эмулятора нажмите зеленую кнопку «Play» рядом с созданным устройством.
4. Запуск приложения
Убедитесь, что эмулятор запущен или подключено реальное устройство. Нажмите зеленую кнопку «Run» (треугольник) на панели инструментов или клавишу Shift+F10. Приложение будет скомпилировано, установлено на устройство и запущено.
5. Редактирование интерфейса (XML)
Откройте файл макета (например, activity_main.xml). Внизу есть два режима: «Design» (визуальный) и «Code» (текстовый). В визуальном режиме можно перетаскивать элементы с панели «Palette» в макет. В текстовом режиме можно вручную редактировать XML-код:
xml
<TextView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hello World!"
    android:textSize="24sp"/>
6. Написание кода активности
Откройте файл MainActivity.java (или .kt). В методе onCreate можно получить доступ к элементам интерфейса:
java
TextView textView = findViewById(R.id.textView);
textView.setText("Новый текст");
7. Добавление библиотек через Gradle
Откройте файл app/build.gradle. В блоке dependencies добавьте нужную библиотеку:
gradle
dependencies {
    implementation 'com.google.android.material:material:1.9.0'
    implementation 'com.squareup.retrofit2:retrofit:2.9.0'
}
После изменения синхронизируйте проект: нажмите «Sync Now» в появившемся уведомлении.
8. Отладка приложения
Установите точку останова, кликнув по полю слева от номера строки. Запустите приложение в режиме отладки (кнопка «Debug» или Shift+F9). При достижении точки останова выполнение приостановится, и вы сможете просматривать значения переменных, выполнять код по шагам (F8 — шаг, F7 — войти в метод).
9. Создание APK (установочного файла)
В меню выберите «Build» → «Build Bundle(s) / APK(s)» → «Build APK(s)». После завершения сборки APK будет находиться в папке app/build/outputs/apk/debug/. Его можно установить на любое Android-устройство.
10. Использование Git в Android Studio
На панели инструментов есть кнопка «VCS». Выберите «Enable Version Control Integration» → «Git». После этого можно коммитить изменения: VCS → Commit, пушить: VCS → Git → Push.
11. Горячие клавиши
•	Ctrl+N — быстрый поиск класса
•	Ctrl+Shift+N — быстрый поиск файла
•	Alt+Enter — автоматические исправления (особенно полезно для импорта библиотек)
•	Ctrl+Alt+L — форматирование кода
•	Ctrl+/ — закомментировать строку
