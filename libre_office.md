7. LibreOffice (офисный пакет)
РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Наименование ПО: LibreOffice (актуальная версия)
Краткое описание
LibreOffice — это бесплатный офисный пакет с открытым исходным кодом, который является полноценной альтернативой Microsoft Office. Включает в себя текстовый процессор Writer, табличный редактор Calc, программу для презентаций Impress, редактор формул Math, систему управления базами данных Base и редактор векторной графики Draw. Пакет поддерживает открытые форматы (ODF) и отлично работает с файлами Microsoft Office (DOCX, XLSX, PPTX), что делает его удобным для совместной работы в смешанных средах. Интерфейс интуитивно понятен и похож на классические версии MS Office.
Запуск программы
Каждое приложение запускается отдельно:
•	Writer (текстовый процессор): libreoffice --writer или через меню «Офис» → «LibreOffice Writer»
•	Calc (таблицы): libreoffice --calc или «Офис» → «LibreOffice Calc»
•	Impress (презентации): `libreoffice --impress» или «Офис» → «LibreOffice Impress»
•	Draw (векторная графика): `libreoffice --draw» или «Офис» → «LibreOffice Draw»
•	Base (базы данных): `libreoffice --base» или «Офис» → «LibreOffice Base»
Также можно просто открыть любой файл двойным кликом — LibreOffice определит тип и запустит соответствующее приложение.
Запланированный выход
•	Горячие клавиши: Ctrl+Q (закрыть текущее приложение) или Ctrl+W (закрыть текущий документ)
•	Через меню: «Файл» → «Выход»
•	При наличии несохраненных изменений программа предложит сохранить документ
•	Закрыть окно (крестик) — также завершит работу
Основные функции
Writer (текстовый редактор):
1.	Создание и редактирование текстовых документов с форматированием (шрифты, абзацы, стили)
2.	Вставка таблиц, изображений, диаграмм, формул и объектов
3.	Создание оглавлений, сносок, колонтитулов
4.	Проверка орфографии и грамматики
5.	Экспорт в PDF, DOCX, HTML и другие форматы
Calc (электронные таблицы):
1.	Создание таблиц с формулами и функциями (SUM, VLOOKUP, IF, COUNTIF)
2.	Построение диаграмм и графиков различных типов
3.	Сортировка и фильтрация данных
4.	Сводные таблицы для анализа данных
5.	Поддержка макросов и скриптов
Impress (презентации):
1.	Создание слайдов с текстом, изображениями, графиками
2.	Применение шаблонов оформления и переходов между слайдами
3.	Добавление анимации к объектам
4.	Экспорт в PDF или в форматы Microsoft PowerPoint
Draw (векторная графика):
1.	Создание схем, блок-схем, плакатов
2.	Работа с фигурами, линиями, текстом
3.	Группировка объектов и слои
Основные приемы работы
1. Создание нового документа
В любом приложении: «Файл» → «Создать» → «Текстовый документ» (или нажмите Ctrl+N).
2. Сохранение документа
Ctrl+S — сохранить. При первом сохранении появится окно для выбора папки и имени файла. Для выбора формата нажмите на выпадающий список «Тип файла» — можно сохранить в форматах ODT (родной), DOCX (для совместимости с Word), PDF, EPUB.
3. Экспорт в PDF (для любого приложения)
Нажмите кнопку «Экспортировать напрямую как PDF» на панели инструментов или выберите «Файл» → «Экспортировать как» → «Экспортировать как PDF». В появившемся окне можно настроить параметры: диапазон страниц, качество изображений, сжатие. Нажмите «Экспорт».
4. Форматирование текста в Writer
Выделите текст и используйте панель инструментов:
•	Шрифт, размер, жирный/курсив/подчеркнутый
•	Выравнивание (по левому краю, по центру, по правому, по ширине)
•	Нумерованные и маркированные списки
•	Цвет текста и заливка фона
5. Использование стилей в Writer
Стили помогают быстро оформить документ единообразно. Откройте панель «Стили» (F11). Выберите текст, затем кликните по нужному стилю (Заголовок 1, Заголовок 2, Обычный). После применения стилей легко создать автоматическое оглавление: «Вставка» → «Оглавление и указатели» → «Оглавление».
6. Создание таблиц в Calc
Введите данные в ячейки. Для вычисления суммы: =SUM(A1:A10). Для вычисления среднего: =AVERAGE(A1:A10). Формула начинается со знака «=».
7. Построение диаграммы в Calc
Выделите данные, включая заголовки. Нажмите «Вставка» → «Диаграмма». Выберите тип (гистограмма, круговая, линейная). Настройте подписи и легенду.
8. Создание презентации в Impress
После запуска Impress выберите шаблон оформления. Добавьте заголовок на первый слайд. Для добавления нового слайда: «Вставка» → «Слайд» или Ctrl+M. Вставьте текст, изображения, таблицы. Для анимации объектов: выделите объект → «Анимация» → выберите эффект.
9. Совместная работа
LibreOffice позволяет сохранять документы в форматы Microsoft Office (DOCX, XLSX, PPTX), что удобно для обмена с пользователями MS Office. При сохранении выберите соответствующий тип файла в выпадающем списке.
10. Горячие клавиши (общие)
•	Ctrl+N — создать новый документ
•	Ctrl+O — открыть существующий документ
•	Ctrl+S — сохранить
•	Ctrl+Z — отменить действие
•	Ctrl+Y — повторить действие
•	Ctrl+P — печать
•	Ctrl+A — выделить всё
•	Ctrl+F — поиск
•	Ctrl+H — замена
